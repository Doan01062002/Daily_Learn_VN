# Sprint 1: Thiết lập dự án & Lược đồ Cơ sở dữ liệu

- **Thời gian dự kiến**: Sprint 1 (Bắt đầu dự án)
- **Mục tiêu chính**: Khởi tạo cấu trúc dự án Next.js chạy được, cài đặt Prisma ORM và định nghĩa lược đồ database PostgreSQL đầu tiên.

---

## Danh sách công việc (Checklist)

- [ ] Thực hiện khởi tạo dự án Next.js (App Router, TS, Tailwind CSS)
- [ ] Xóa các code mẫu không cần thiết trong Next.js (`page.tsx`, `globals.css`)
- [ ] Thiết lập tài liệu lược đồ database `docs/database_schema.md`
- [ ] Cài đặt Prisma ORM (`prisma`, `@prisma/client`)
- [ ] Định nghĩa các model dữ liệu trong `prisma/schema.prisma`
- [ ] Tạo file `.env` mẫu để cấu hình DATABASE_URL
- [ ] Chạy lệnh `npm run build` và `npx prisma validate` để xác thực dự án
